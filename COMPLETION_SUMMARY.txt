## 🔄 System Flows - SEMUA TERINTEGRASI

### Flow 1: Setoran Sampah → Saldo
```
User Input Waste
    ↓
TrashData (pending)
    ↓
Admin Approve
    ↓
✅ User.saldo += total_price
✅ Gudang inventory updated
✅ Notification sent
```

### Flow 2: Withdrawal Request
```
User Request
    ↓
Deduct saldo (pending)
    ↓
Admin Approve/Reject/Process
    ↓
✅ Update status
✅ Send notification
✅ If reject: refund saldo
```

### Flow 3: Informasi Management
```
Admin Create
    ↓
Save (draft/published)
    ↓
User Browse & Read
    ↓
✅ View counter increment
✅ Related info suggestions
```

### Flow 4: Maggot Cultivation
```
Create Batch
    ↓
Monitor Status
    ↓
Panen & Alokasi
    ↓
✅ Maggot kering → Gudang
✅ Pupuk organik → Gudang
✅ Penjualan langsung
```
